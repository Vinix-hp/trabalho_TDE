# 🐍 Jornada Python: Raciocínio Algorítmico (PUCPR)

Repositório dedicado ao versionamento dos exercícios e desafios desenvolvidos na disciplina de **Raciocínio Algorítmico** (TDE) durante o 1º Semestre de 2026. O foco aqui é o aprendizado de lógica com Python e o domínio de ferramentas de controle de versão como Git e GitHub.

---

## Conteúdo do Repositório

| Arquivo / Pasta | Descrição do Conteúdo |
|:--- |:--- |
| `aula03-introducao.py` | Fundamentos: Variáveis, tipos de dados e entrada/saída |
| `aula04-condicionais.py` | Estruturas de decisão: Lógica com `if`, `elif` e `else` |
| `aula05-listas.py` | Armazenamento de dados: Listas e Vetores |
| `aula06-matrizes.py` | Estruturas multidimensionais e manipulação de tabelas |
| `aula07-dicionarios.py` | Estruturas de Chave-Valor: Dicionários em Python |
| `aula08-funcoes.py` | Modularização: Criação de funções e escopo |
| `lista-Ra1.py` | Exercícios preparatórios para a avaliação RA1 |
| `lista-Ra2.py` | Exercícios avançados e integração com bibliotecas (RA2) |
